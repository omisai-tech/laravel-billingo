# Changelog

All notable changes to `laravel-billingo` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).


## [1.0.0] - 2025.11.30

### Added

- Initial release
- Full Billingo API v3 support
- Laravel 10, 11, and 12 compatibility
- PHP 8.2+ support
- Service Provider with auto-discovery
- Facade for easy API access
- Dependency Injection support for all API classes
- Complete model classes generated from OpenAPI specification
- Comprehensive test suite (250+ tests)
- Full documentation with usage examples

### API Support

- **Partner API** - Create, read, update, delete partners
- **Product API** - Manage products and services
- **Document API** - Create invoices, proformas, receipts, and more
- **Bank Account API** - Manage bank accounts
- **Document Block API** - Invoice numbering sequences
- **Currency API** - Currency conversion rates
- **Organization API** - Organization settings and data
- **Spending API** - Track and manage expenses
- **Document Export API** - Bulk export documents
- **Utility API** - Tax number validation and more
